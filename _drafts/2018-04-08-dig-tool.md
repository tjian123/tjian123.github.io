> `dig` 是一个通过查询 DNS 服务器来获取主机地址、邮件交换等信息的命令行工具。

<!--more-->

最简单的用法：

	dig www.baidu.com

输出为：


`dig` 支持的 DNS 查询包括：

- A，IP 地址
- TXT，文本注解
- MX，邮件交换
- NS，name server
- ANY，以上所有类型

此外，还有一些格式化选项：
- +short，简化输出
